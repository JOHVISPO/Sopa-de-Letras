Instrucciones de Uso - Sopa de Letras

Descripción:
Este proyecto consiste en una sopa de letras implementada en HTML, diseñada para que los usuarios busquen palabras específicas en una cuadrícula de letras.

Instrucciones de Uso:
1. Descarga este repositorio en tu máquina local.
2. Abre el archivo `index.html` en un navegador web compatible.
3. Una vez que el proyecto se abra en el navegador, verás una cuadrícula de letras.
4. Busca palabras en la sopa de letras seleccionando las letras en orden consecutivo. Puedes seleccionar las letras haciendo clic en ellas y al hacerle doble click sobre la misma se desmarcará.
5. Cuando hayas seleccionado una palabra completa, se resaltará y recibirás una notificación de que la palabra ha sido encontrada.
6. Continúa buscando palabras hasta que hayas encontrado todas las palabras especificadas.
7. Una vez que hayas encontrado todas las palabras, recibirás una notificación de felicitaciones.

Definiciones:
En la parte derecha de la página, encontrarás las palabras que tienes que buscar con sus respectivas definiciones, relacionadas con el desarrollo de software.


¡Disfruta buscando palabras en la sopa de letras!
